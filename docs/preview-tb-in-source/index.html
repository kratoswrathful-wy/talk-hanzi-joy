SELECTED

極淡標示 + 術語旁上標數字 + 副行對照（跟著右欄分頁同步）

示意：同一術語在句中重複出現會淡標示多次，且每次出現都顯示上標（更直覺知道該按哪個 Ctrl+數字）。
