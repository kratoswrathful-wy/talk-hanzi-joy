                快捷鍵

                固定快捷鍵

                下列為編輯檢視（已開啟檔案）內預設可用的快捷鍵。標示為 Ctrl 者請按鍵盤上的 Control（Mac 上通常為「control」，不是 ⌘ Command）；標示為 Alt 者在 Mac 上多為 Option。

                確認／取消確認句段

                焦點在該句「譯文」欄時：若為未確認會寫入並標為已確認（可依設定跳到下一句或連動重複句段）。搭配多選句段時可批次確認。

                跳到上一／下一句段

                焦點在譯文欄時，移動到可見列表中上一則或下一則可編輯句段的譯文開頭（略過篩選隱藏列）。

                全選目前可見句段

                在目前搜尋／篩選結果中可見的句段全部勾進「選取」（# 欄會顯示選取框線），方便批次取代或批次確認。

                插入下一個缺漏標籤

                在譯文中依標籤順序插入下一組缺的占位標籤（有成對規則時可一次補齊）。

                點擊原文標籤插入譯文

                在「原文」欄點擊某顆標籤，將該佔位符插入同一句譯文：若本句曾留下暫存游標則插在該處，否則插在譯文末尾。譯文已含該佔位符則不重複插入。句段鎖定或他人協作編輯中時不會插入。

                標籤展開／收起顯示

                切換標籤在編輯區是完整展開或收成較短樣式；僅影響畫面呈現，不刪除標籤內容。實作以鍵盤實體 S 鍵偵測，注音等鍵盤配置下仍可用。

                清除譯文中所有標籤占位

                移除譯文裡形如 {數字} 的標籤碼，保留可見文字。

                將原文複製到譯文

                對選取的句段（或多選）把原文拷貝到對應譯文欄，沿用標籤對應規則。

                清除譯文

                對選取的句段清空譯文（已確認者會改為未確認）；請留意無法編輯的鎖定句段會略過。

                套用右側第 N 筆建議（依分頁）

                右側在「CAT」分頁且已載入比對建議時，套用到目前句段；在「TM 搜尋」分頁時，套用目前 concordance 列表第 1～9 筆。

                插入選取文字到譯文游標

                將目前畫面上任意反白的文字插入「譯文」中先前記住的游標位置（或作用中的譯文格）；常用於從 CAT／別列複製片段後貼入。

                復原／重做

                在編輯器內復原或重做譯文變更、確認狀態等可追蹤的步驟。焦點在尋找／取代欄或 TM 搜尋關鍵字輸入框時不觸發，避免與瀏覽器或輸入框預設行為衝突。

                在比對列表內移動選取（CAT／TM 搜尋）

                右欄在「CAT」分頁（比對建議表）或「TM 搜尋」分頁（concordance 列表）且已載入列表時，上移／下移目前選中列，便於檢視 metadata；不會套用譯文（雙擊或 Ctrl+1～9 才套用）。

                CAT 結果換頁

                在「CAT」分頁、比對結果多於 9 筆時，向左／向右換頁。在譯文／原文可編格內不觸發（保留 Alt+←／→ 的預設行為或格內導向）。

                右欄分頁切換

                在 CAT、TM 搜尋、新增術語、QA 等分頁間循環切換，不佔用僅帶 Ctrl 的左右鍵（格內逐字）。

                譯文游標／假游標（捲動與還原）

                上下鍵效果相同：若焦點已在譯文格，捲動使該格進入可視區；否則嘗試還原先前記住的譯文游標，失敗則顯示假游標（垂直條提示）。

                譯文格：上／下鍵越句

                僅在譯文格內、不加 Ctrl。游標在句段內第一可視行再按上，到上一可見句譯文開頭；在最末可視行再按下，到下一可見句譯文開頭；篩選隱藏之列略過。

                尋找／篩選列

                第一次：將焦點移到左上「尋找」欄以便輸入或取代；若有反白文字會帶入尋找框。焦點已在尋找欄時再按：在「搜尋」與「篩選」模式間切換（進階篩選鎖定時可能無法切換）。部分情境下 QA 範圍鎖定篩選時會無法使用。

                搜尋下一個／上一個命中

                與工具列「下一個／上一個」相同：依目前搜尋條件與勾選範圍（原文／譯文／Key／額外等）在全表中循環導覽命中。須已開啟檔案且編輯檢視未隱藏；不加 Ctrl／Alt。

                全部取代（譯文）

                等同按下「全部取代」：僅替換各句譯文欄中符合尋找字串（含 regex 選項）的文字；僅處理篩選下可見句段；若已多選句段則僅限選取範圍。須已開啟檔案且編輯檢視未隱藏；不加修飾鍵。

                TM 一致性搜尋

                切到「TM 搜尋」分頁。有反白字時帶入關鍵字並搜尋專案掛載的 TM 快取；可選「Key」以 key 欄內文比對；在 Key 格內觸發時會搜尋 Key 且焦點到譯文句首。譯文欄內有反白時，搜尋後不保留反白、僅留游標。無反白時不搜尋，僅關注鍵字欄。

                關閉最上層蓋版

                關閉目前最上層的對話／蓋版（等同取消或關閉鈕）。錄製自訂快捷鍵過程中按下 Esc 為中止錄製、不關閉快捷鍵視窗。

                可自訂快捷鍵

                點「錄製」後按下任意組合鍵即可設定；「重設」恢復預設值。
